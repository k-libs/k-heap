class Heap<T>

Heap

A simple, generic heap implementation that uses a caller provided function to enable heaps of custom types.

The basic heap types, min-heap and max-heap, can be implemented by using a simple lambda such as:

// min heap
{ a, b -> a < b }

// max heap
{ a, b -> a b }
