fun <T> Heap(initialCapacity: Int = 8, scaleFactor: Float = 1.5f, maxSize: Int = Int.MAX_VALUE, sortTest: SortTest<T>)

Constructs a new heap instance.

Parameters

initialCapacity

Initial capacity for the data container underlying the new heap.

If this value is greater than maxSize, an exception will be thrown.

Default: 8

scaleFactor

How quickly the capacity will scale.

Default: 1.5

maxSize

Maximum allowed size for this heap.

Default: 2147483647

sortTest

Function used to test and verify the ordering of the values in this heap.

Throws

IllegalArgumentException

If the given initialCapacity value is greater than the given maxSize value.
